# T-IME 통합 그룹웨어

React로 개발된 통합 그룹웨어 시스템입니다.

## 기능

- 📊 대시보드 - 전체 현황 및 빠른 작업
- 🏢 회의실 예약 - 회의실 예약 및 관리
- 🚗 차량 예약 - 법인 차량 예약 및 관리
- 💐 경조사 관리 - 경조사 신청 및 관리
- 📺 전광판 관리 - 디지털 사이니지 콘텐츠 관리
- 📈 통계 - 시설 이용 현황 및 통계
- ⚙️ 관리자 설정 - 시스템 및 사용자 관리

## 기술 스택

- React 18
- React Router DOM 6
- Tailwind CSS 3
- Vite

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

개발 서버가 실행되면 브라우저에서 `http://localhost:5173`으로 접속할 수 있습니다.

### 빌드

```bash
npm run build
```

### 미리보기

```bash
npm run preview
```

## 프로젝트 구조

```
src/
  ├── components/     # 공통 컴포넌트
  ├── layouts/        # 레이아웃 컴포넌트
  ├── pages/          # 페이지 컴포넌트
  └── utils/          # 유틸리티 함수
```
